# Spektif Agency

A modern project management application optimized for Turkish users, built with Next.js frontend and Firebase backend.

## Current Architecture

- **Frontend**: Next.js 14 with TypeScript (Vercel)
- **Backend**: Firebase Functions (europe-west4)
- **Database**: Firebase Firestore (europe-west4)
- **Authentication**: Firebase Auth + NextAuth.js
- **Storage**: Firebase Storage

## Quick Start

```bash
# Install dependencies
bun install

# Start development
bun dev

# Deploy functions
firebase deploy --only functions
```

## Performance

- **Database Latency**: 20-30ms (optimized for Turkey)
- **API Response Time**: 200-500ms
- **Region**: europe-west4 (Belgium) - closest to Turkey
